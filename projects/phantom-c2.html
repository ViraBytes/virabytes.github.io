Phantom C2

Undetectable command-and-control framework with built-in EDR evasion

Red Team C2 Framework 2025 Python

Project Overview

Phantom C2 is an advanced command-and-control framework designed to bypass modern EDR solutions using cutting-edge evasion techniques. Developed during real-world red team operations, it incorporates proprietary methods to maintain stealth while providing full control over compromised systems.

The framework has been tested against industry-leading security products including CrowdStrike Falcon and SentinelOne, demonstrating consistent evasion capabilities even in monitored enterprise environments.

Technical Highlights

Process Hollowing

Advanced code injection technique that executes payloads within legitimate process memory space, bypassing signature-based detection.

API Unhooking

Dynamic resolution of clean API copies from ntdll.dll to evade user-space hooks placed by security products.

Sleep Obfuscation

Custom sleep encryption that hides beaconing patterns from network traffic analysis tools.

Modular Architecture

Plugin-based system allowing operators to extend functionality without modifying core components.

Technology Stack

Python 3.10 Asyncio WebSockets AES-256 Rust (for critical components)

Key Challenges

  • EDR Evasion: Developed custom techniques to bypass memory scanning and behavioral detection
  • Network Stealth: Implemented domain fronting and encrypted C2 channels
  • Payload Generation: Created polymorphic payload engine to avoid static signatures
  • Operational Security: Designed automatic cleanup procedures to reduce forensic artifacts
